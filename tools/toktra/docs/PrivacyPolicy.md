# toktra Privacy Policy

Last updated: 2026-05-15

toktra is a Chrome extension that translates English webpages, selected text, and PDF content into Simplified Chinese.

## Data Processed

toktra may process the following data:

- English text from webpages, selected text, and PDFs that the user chooses to translate.
- Page URL or domain for per-site auto-translation settings, local cache keys, and cached webpage-structure strategies.
- API configuration entered by the user, including API base URL, model, and API key.
- Local translation cache and local webpage-structure strategy cache.

## How Data Is Used

Text is used only to request translation from the OpenAI-compatible API endpoint configured by the user. If smart structure analysis is enabled, toktra sends a lightweight page outline to the configured API so it can identify readable page regions. This outline may include the page title, language, candidate selectors, text lengths, and short text samples. toktra does not send full page HTML for structure analysis.

The API key is used only to authorize requests to the configured API endpoint.

## Storage

toktra stores settings, the API key, translation cache, and site strategies in Chrome extension local storage on the user's device. The extension does not use an external toktra server.

## Data Sharing

toktra sends translation requests only to the API endpoint configured by the user. toktra does not sell user data, does not use user data for advertising, and does not include analytics tracking.

Users should review the privacy policy and retention behavior of their configured translation API provider because that provider receives the text submitted for translation.

## User Control

Users can:

- Disable translation for the current page.
- Disable automatic translation globally.
- Enable automatic translation only for selected domains.
- Clear or overwrite API settings from the extension options page.
- Remove the extension to delete its locally stored extension data according to Chrome's extension storage behavior.

## Contact

For questions about this extension, use the support/contact channel listed on the Chrome Web Store listing.
